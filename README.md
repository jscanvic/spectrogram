# Usage

Clone the repository

```bash
git clone https://github.com/jscanvic/spectrogram
cd spectrogram
```

Start an HTTP server in the current directory

```bash
python -m http.server -b 127.0.0.1 8000
```

Open [this URL](http://127.0.0.1:8000/) in a web browser
